Timeline elements

  • Step execution - Step execution is displayed as a solid colored rectangle inside the timeline. Step execution starts at the rectangle start and finishes / pauses / continues at the rectangle end.
  • Stage execution - Stage execution is displayed as a transparent rectangle inside the timeline. Each stage is labeled with a label on the left side of the timeline. Steps belonging to a stage are displayed inside it's body.
  • Execution - Execution start and finish times are displayed as a solid vertical red line.
  • Coloring - Each stage / step is colored according to it's state as shown in the color legend section.

Controls

  • Scroll - Moves the report execution up / down.
  • Drag - Moves the timeline left / right.
  • Tick size picker:
    • Tick size - Sets the duration of single tick on the timeline.
    • Tick unit - Sets the unit of time for tick size.
  • Click on the step / stage label - Displays execution detail next to the tick size picker.
  • Paginator - You can switch between run executions with paginator arrows.
