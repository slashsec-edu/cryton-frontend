Stage creation help page

How to create a stage?

  1. Fill out stage name (stage name must be unique in the template).
  2. Select trigger type, delta trigger is chosen by default.
  3. Fill out needed trigger arguments (trigger arguments will change according to the trigger type)
  4. Create steps in the "Create step" tab. Steps will appear in the "Show dependency tree" section after creation.
  5. Create a dependency tree (more info in Show dependency tree > Help):
    1. Click on "Show dependency tree" button to move to dependency tree creation.
    2. Move steps you want to use from the dispenser into the dependency tree.
    3. Create all step dependencies you want. There must be exactly one node without a parent dependency (root node).
    4. Dialog for conditions will appear after creating a dependency. A dependency has to have at least one condition, otherwise it won't be created.
    5. You can add as many conditions as you like with "Add condition" button.
  6. Click on "Create stage" button to create the stage.
  7. Stage will now appear in Build template > Show dependency tree

Tips

  • If the "Create stage" button is disabled hover over it and it will show you an error message
  • You can see the dependency tree creation progress on the right side of the "Create stage" tab. (Only visible for screens wider than 1600px)
  • If you mess up you can erase stage parameters form with "Erase parameters" button.
