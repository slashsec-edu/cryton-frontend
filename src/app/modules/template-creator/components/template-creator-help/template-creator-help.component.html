Template creation help page

How to create a template?

  1. Fill out template name and owner.
  2. Create all of the neccessary stages in the "Create stage" tab (More info in Create stage > help).
  3. Create a dependency tree (more info in Show dependency tree > Help):
    1. Click on "Show dependency tree" button to move to dependency tree creation section.
    2. Move all stages you want to use from the stage dispenser into the plan.
    3. Create dependencies between stages (there can be multiple root nodes unlike in stage creation).
  4. Click on "Create template" button to create template.
  5. Template will be now created in the cryton core and available throughout the app.

Timeline

  • You can optionally manage stage start times in the "Show timeline" section.
  • You can adjust start time of delta trigger stage by dragging it along the timeline.
  • More info in Show timeline > Help.
