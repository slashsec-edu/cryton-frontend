Help

Timeline elements

  • Stages
    • Only stages with trigger type of delta are displayed inside the timeline.
    • Stage is displayed as a circle inside the timeline with a label which contains its name.
    • Stages are placed at their execution start time and can be moved around by dragging, changing stage position will change its execution start time.
    • Stage execution start time gets displayed when moving a node at the top of the timeline.
  • Stage dependencies
    • Dependent stages are connected with an arrow.
    • Transitive dependencies are shown as well, for example with a dependency tree like this: D -> N -> D, the timeline will display D -> D. (D = Delta stage, N = Non-delta stage)
    • Dependecies will get colored to red if a stage has a greater delta than his parent.

Controls

  • Tick size picker:
    • Tick size - Sets the duration of single tick on the timeline.
    • Tick unit - Sets the unit of time for tick size.
  • Scroll - Moves the template up / down.
  • Drag - Moves the timeline left / right.
  • Double click on stage - Opens stage settings.
  • Shift + click stage - Selects / unselects a stage, all selected stages can be then moved together by moving any of the selected nodes. Click on the timeline to unselect all stages.

Toolbar

  • swap_vertEnables vertical movement of stages.
  • shareWhen moving a stage, also moves all of its children.
  • auto_awesomeAutomatically organizes the timeline.
