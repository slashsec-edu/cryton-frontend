Welcome to template creator!

This page will give you an outline of the process of creating a plan template. Further information can be found in help pages for individual creation steps.


1. Create steps

In this step you create steps in the attack plan which will be later used to create attack stages. You can find the attack steps that you create in the next creation step ("Create stages" > "Show dependency graph").

2. Create stages

In this step you create attack stages which will be finally used to put together a plan template. Each stage contains a set of steps with some optional dependencies and defines a trigger which controls when the stage will start. The trigger can be defined in the stage parameters form. To create a dependency graph navigate to the "Create stages" > "Show dependency graph". You can find created stages in the next step ("Create template" > "Show dependency graph").

3. Create template

This is the final part of plan template creation, here you can create optional dependencies between the stages and edit plan's timeline. After you are happy with the result you can click on "Create template" button which will show you the final template's YAML which you can edit before creation.
