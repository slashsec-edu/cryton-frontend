Dependency graph help page

Components

  • Item dispenser - Contains created nodes, nodes can be moved to the plan with swap_horizontal_circle.
  • Dependency graph - Used to create dependencies between nodes.

Toolbar

  • zoom_in Zooms in the dependency graph by the factor of 0.1.
  • zoom_out Zooms out the dependency graph by the factor of 0.1.
    • Graph has a scale of 1 by default and can be scaled down to 0.1 or up do 1.5.
  • fullscreen Fits and centers the dependency graph inside the graph nicely.
  • back_hand Enables dragging nodes.
  • swap_horizontal Enables swapping nodes back into the dispenser (nodes can be moved back afterwards).
  • delete_forever Enables deleting nodes and edges.

Manipulation

  • You can move the whole graph by dragging by default.
  • You can move individual nodes by enabling back_hand and dragging.
  • You can swap nodes back into the dispenser by enabling swap_horizontal and clicking on the node (moving dash should appear around the node on hover).
  • You can delete nodes forever by enabling delete_forever and clicking on the node (moving dash should appear around the node on hover).
  • You can edit a node by clicking on the settings.
  • Edge creation:
    • Click on the colorful dot on the bottom of the node to create an edge.
    • You can destroy dragged edge by clicking away from the nodes.
    • Connect the edge by clicking on another node.
    • You can destroy created edge by enabling delete_forever and clicking on the edge (moving dash should appear on hover).
