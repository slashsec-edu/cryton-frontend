Stage creation help page

How to create a stage?

  1. Fill out stage name (stage name must be unique in the template).
  2. Select trigger type, delta trigger is chosen by default.
  3. Fill out needed trigger arguments (trigger arguments will change according to the trigger type).
  4. Create a dependency graph (more info in Show dependency graph > Help):
    1. If you haven't already, create steps in the "Create step" tab. Steps will appear in the "Show dependency graph" section after creation.
    2. Click on "Show dependency graph" button to move to dependency graph creation.
    3. Move steps you want to use from the dispenser into the dependency graph.
    4. Create needed step dependencies by clicking on the red circle of the parent step and then on the child step.
    5. Dialog for conditions will appear after creating a dependency. A dependency has to have at least one condition, otherwise it won't be created.
    6. You can add as many conditions as you like with "Add condition" button.
  5. Click on "Create stage" button to create the stage.
  6. Stage will now appear in Build template > Show dependency graph, inside a stage dispenser. If you don't use a stage, it will stay in the dispenser even after creating a template so you can use it in the next template.

Tips

  • If the "Create stage" button is disabled hover over it and it will show you an error message.
  • You can see the dependency graph creation progress on the right side of the "Create stage" tab. (Only visible for screens wider than 1600px)
  • If you mess up you can erase stage parameters form with "Erase parameters" button.
