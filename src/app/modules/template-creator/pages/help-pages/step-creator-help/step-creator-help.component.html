Step creation help page

How to create a step?

  1. Fill out name (step name must be unique in stage that is currently being created).
  2. Fill out attack module.
  3. Fill out attack module arguments in YAML format.
  4. Click on "Create step" button to create the step.
  5. Step will now appear in Create stage > Show dependency tree, inside a step dispenser. If you don't use a step, it will stay in the dispenser even after creating a stage so you can use it in the next stage.

Tips

  • Using TAB in attack module args textarea will insert 2 spaces.
  • If the "Create step" button is disabled hover over it and it will show you an error message.
  • If you mess up, you can quickly erase the whole form with "Erase parameters" button.
