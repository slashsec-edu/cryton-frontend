Template creation help page

How to create a template?

  1. Fill out template's name and owner.
  2. Create a dependency graph (more info in Show dependency graph > Help):
    1. If you haven't already, create all of the neccessary stages in the "Create stage" tab (More info in Create stage > help).
    2. Click on "Show dependency graph" button to move to dependency graph creation section.
    3. Move all stages you want to use from the stage dispenser into the plan.
    4. Create dependencies between stages (there can be multiple root nodes unlike in stage creation).
  3. Click on "Create template" button to create template.
  4. Template will be now created in the cryton core and available throughout the app.

Timeline

  • You can optionally manage stage start times in the "Show timeline" section.
  • You can adjust start time of delta trigger stage by dragging it along the timeline.
  • More info in Show timeline > Help.
